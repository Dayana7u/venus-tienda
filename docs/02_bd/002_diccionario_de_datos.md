# Diccionario de datos inicial

## Trazabilidad obligatoria
Todas las tablas base del proyecto manejan:
- estado
- borrado
- usuario_creacion
- fecha_creacion
- usuario_modificacion
- fecha_modificacion
- usuario_borrado
- fecha_borrado

## Tablas base revisadas para parametrización
### system.temas
Registra los temas visuales disponibles.

### system.tema_tokens
Registra tokens configurables por tema.

### system.tema_componentes
Registra configuración visual por componente.

### system.branding
Registra identidad visual general.

### system.parametro_grupos
Clasifica parámetros por grupo funcional.

### system.parametros
Registra catálogo maestro de parámetros.

### system.parametro_valores
Registra valores configurables de parámetros.

### system.modulos
Registra módulos habilitables.

### system.modulo_configuraciones
Registra configuración por módulo.

### system.integraciones
Registra integraciones disponibles.

### system.integracion_configuraciones
Registra configuración técnica por integración.

### system.plantillas
Registra plantillas reutilizables.

### system.menus
Registra configuración de menús.
