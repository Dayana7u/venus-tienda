# Estándares SQL aplicados

- Todas las consultas deben incluir esquema.
- Las palabras reservadas van en mayúscula sostenida.
- La tabulación se maneja a dos espacios.
- Las consultas deben validar estado y borrado cuando aplique.
- Los ORDER BY deben declararse de forma consistente.
- La trazabilidad documental del SQL debe mantenerse junto al avance funcional.
